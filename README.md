# EduConnect-Bridging-the-Learning-Gap-with-AI
An AI-driven educational platform built to support students with visual, hearing, and learning disabilities. It offers real-time tools such as speech-to-text, text simplification, gamified learning, and Android accessibility — all designed to make education more inclusive and personalized.

1. Problem Statement:

Many students with disabilities encounter barriers in traditional classrooms due to the absence of assistive tools.

Visually impaired students require audio output to understand written content.

Hearing-impaired students need real-time text or visual translation of speech.

Dyslexic learners find it difficult to comprehend complex written language.

This project addresses these issues by providing a unified, AI-powered learning hub tailored to each student’s needs.

2. Features:

a)Speech-to-Text
Converts spoken content into written text for hearing-impaired learners using OpenAI Whisper.

b)Text-to-Speech
Converts written content into audio to support visually impaired learners using Google TTS or pyttsx3.

c)Text Simplification
Uses NLP with the T5 transformer to simplify complex text into easier language for better comprehension.

d)Gamified Learning
Interactive quizzes with rewards to improve engagement and retention.

e)Android Application
A mobile-friendly interface to access all features anytime, anywhere.

3.Tech Stack:

Languages and Frameworks: Python, Streamlit, Flask, Android (Kivy or Flutter)

4.Libraries and Models:

OpenAI Whisper – Speech recognition

Google TTS / pyttsx3 – Text-to-speech

Hugging Face T5 – Text simplification
